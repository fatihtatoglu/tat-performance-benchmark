# Fibonacci Sequence

A simple implementation of the Fibonacci sequence calculation was made by recursive. The formulation was received from Wikipedia.

## Formula

$$F_0 = 0, F_1 = 1$$

and

$$F_n = F_{n-1} + F_{n-2}$$

for $$n > 1$$
